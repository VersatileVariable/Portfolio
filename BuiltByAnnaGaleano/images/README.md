# Images Directory

## Profile Images
Place your profile picture in the `profile/` directory. The recommended file name is `anna-profile.jpg` or `anna-profile.png`.

Recommended dimensions: 400x400px (square format works best)

## Gallery Images
Place career-related photos in the `gallery/` directory. These could include:
- Conference presentations
- Lab work photos
- Project demonstrations
- Team photos
- Award ceremonies
- Workshop participation

Recommended format: JPG or PNG
Recommended size: Under 2MB per image

## Usage in HTML
The HTML is set up to use:
- Profile image: `images/profile/anna-profile.jpg`
- Gallery images: `images/gallery/` with descriptive filenames

## Image Optimization
For best performance, consider:
- Compressing images before upload
- Using appropriate file formats (JPG for photos, PNG for graphics with transparency)
- Including alt text for accessibility
